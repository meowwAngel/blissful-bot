# Install guide
1. Clone the repo
2. Head to https://discord.com/developers/ and make a new bot, go to the bot and select the "bot" tab
<img width="1909" height="903" alt="image" src="https://github.com/user-attachments/assets/2a280f95-69dc-484f-b196-712110ad94a3" />
3. Click on reset token and copy it
4. Paste the token into .env variable
5. Edit it how you want, I recommend watching some videos on how to use python.py or look at some reddit threads
6. Have fun :3
